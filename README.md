## Цель работы:

Изучить жизненный цикл Activity в Android и научиться правильно обрабатывать переходы между состояниями. В результате лабораторной работы будет создано интерактивное приложение "Grade Clicker" (Кликер оценок), которое демонстрирует работу lifecycle callbacks и сохранение состояния при изменении конфигурации устройства.
  
___
## Теоретическая часть

**Activity Lifecycle (Жизненный цикл Activity)** — это последовательность
состояний, через которые проходит Activity от момента создания до уничтожения.
В течение своей жизни Activity переходит между различными состояниями. Эти
переходы называются жизненным циклом Activity.

## Основные компоненты Activity Lifecycle

* Created — Activity создана, но ещё не видна
* Started — Activity видна, но не в фокусе
* Resumed — Activity на переднем плане и активна
* Paused — Activity частично видна (например, диалог поверх)
* Stopped — Activity не видна (пользователь перешёл в другое приложение)
* Destroyed — Activity уничтожена, ресурсы освобождены
